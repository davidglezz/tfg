## Instalar

1. Install all dependencies:

    `npm install`
    
2. Run the project:

    `npm start`

3. Open in browser:
 
    `http://localhost/`

### Using with older versions of node

This project targets ES6. 
You can target ES5, but you'll need to use es6-shim and install its typings.
